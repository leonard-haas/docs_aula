# aula Docs

## Prerequisites

Install:

- If using `python`'s `pip`: `pip install mkdocs pymdown-extensions mkdocs-material mdx_truly_sane_lists`
- Otherwise, install packages: `mkdocs python-pymdown-extensions mkdocs-material python-mdx_truly_sane_lists`

## Running

```bash
mkdocs serve
```

## Deploying

```bash
mkdocs gh-deploy
```
